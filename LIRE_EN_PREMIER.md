## 🆘 Besoin d'Aide ?

### Problèmes Courants

**❌ "Invalid API key"**
→ Vérifiez votre fichier `.env.local`

**❌ "relation does not exist"**
→ Exécutez le fichier `supabase/schema.sql`

**❌ Impossible de se connecter**
→ Vérifiez que l'utilisateur existe dans Supabase
